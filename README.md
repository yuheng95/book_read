# SG-Korpus — 每日德语背诵语料

> 👉 **新手先看 [`使用说明.md`](使用说明.md)。** 每天同步今天的新语料：
> ```powershell
> git -C "C:\Users\Yuhen\Desktop\de_ex\slow-german" pull
> ```

受 [Slow German](https://slowgerman.com/) 启发的 **B2 中高级** 德语背诵材料。
每天一篇 markdown：**情景对话 + 仿播客短文 + 中英双语词汇 + 句型 + 语法点 + 背诵清单**。

> ⚠️ 所有德语文本为**原创**，话题取材自 Slow German 的选题方向，**不照抄其版权文稿**。
> 原版免费文稿/音频请直接到 [slowgerman.com](https://slowgerman.com/) 听读。

## 配置（本系列固定）
- **水平：** B2
- **形式：** 对话 + 短文 混合
- **标注：** 中英双语

## 文件命名
`YYYY-MM-DD_Thema.md`，例如 `2026-06-05_Muelltrennung.md`

## 每篇结构
1. **Teil 1 · Dialog** — 双人情景对话（可角色扮演）
2. **Teil 2 · Kurztext** — 第一人称/说明性短文
3. **Wortschatz** — 词汇表（德 / 词性 / 中 / 英）
4. **Redewendungen & Satzmuster** — 重点表达与句型
5. **Grammatik-Fokus** — 当天一个语法点 + 例句
6. **背诵清单** — 6~8 句脱稿目标，可打勾

## 如何继续
每天对我说一句即可，例如：
- 「**生成今天的语料**」（我会自动选下一个话题）
- 「**今天我想背 _Oktoberfest_**」（指定话题）
- 「**换个更口语的话题**」/「**今天多放点语法**」（调整风格）

也可以让我**自动化每天生成**（用 `/schedule` 定时，或 `/loop`）——需要的话告诉我。
